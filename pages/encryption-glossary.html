Glossary of Encryption Terminology

Welcome to the Encrypti0n.com glossary: Your quick, concise reference to essential cryptographic and encryption terms clearly explained for easy understanding.

AES (Advanced Encryption Standard)

A widely adopted symmetric encryption algorithm used globally to securely encrypt data. Encrypti0n.com specifically uses AES-GCM.

AES-GCM (Galois/Counter Mode)

An advanced encryption mode of AES providing both confidentiality and built-in authentication, protecting against data tampering.

Base64

A method for encoding binary data into ASCII text, commonly used to safely transmit encrypted data.

Ciphertext

Data that has been transformed from readable plaintext into encrypted form, unreadable without the correct decryption key.

Encryption

The secure transformation of readable data (plaintext) into an unreadable format (ciphertext) to prevent unauthorized access.

Decryption

The process of converting encrypted data (ciphertext) back into its original readable form (plaintext).

IV (Initialization Vector)

A random, unique value used during encryption to ensure that identical plaintext encrypted multiple times yields different ciphertext.

Nonce

Short for "number used once," a unique, random value used to protect encryption against replay attacks and ensure security in AES-GCM encryption.

PBKDF2 (Password-Based Key Derivation Function 2)

A cryptographic method that securely generates encryption keys from user passwords by using salt and multiple hashing rounds, making keys resistant to brute-force attacks.

Plaintext

Unencrypted data in its original, readable format before undergoing encryption.

Salt

Random data added during key derivation processes (like PBKDF2) to protect against precomputed attacks and ensure each encryption key is unique.

Symmetric Encryption

A type of encryption where the same cryptographic key is used for both encrypting and decrypting data.

Tamper-proof

Characteristic of encrypted data (like AES-GCM) that ensures any alteration or corruption during storage or transmission can be immediately detected upon decryption.

Web Crypto API

A secure browser-based JavaScript API providing robust cryptographic operations, such as AES-GCM encryption and PBKDF2 key derivation, used extensively by Encrypti0n.com.
